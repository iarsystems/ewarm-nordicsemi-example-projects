Example description

  
Example project to show gpiote functionality

DESCRIPTION
===========
 
  This example shows the usage of GPIOTE peripheral. We use TIMER0 to generate 'EVENT_COMPARE0' event every
 200 milliseconds. This will also clear the TIMER0 internal clock. This event is passed through PPI peripheral
 to generate a task. This task is used by GPIOTE (task mode) to toggle the pin. This means that every time we
 have EVENT_COMPARE event, the GPIO pin toggles automatically, which can only be achieved using the GPIOTE
 peripheral.
 [TIMER0]->EVENT_COMPARE->[PPI]->GPIOTE_TASK_OUT->[GPIOTE]->toggle_pin
 
  For more detailed information see nRF51 SDK.

COMPATIBILITY
=============

   The example project is compatible with, on Nordic Semi nRF6310 - PCA10006 evaluation
  boards. By default, the project is configured to use the J-Link SWD interface.

CONFIGURATION
=============

   Make sure that the following jumpers and swiches are correctly configured on the
  nRF6310 evaluation board:
  Switches:
  S8  - VBUS
  S9  - ON
  S11 - ON
  
  PCA10006 board:
  Jumpers:
   K1 - 2-3
   P4 - 1-3 2-4 (32 kHz crystal)

  The Blinky application is downloaded to the iFlash.

GETTING STARTED
===============

  1) Build and download the program.

  2) Run the program.
