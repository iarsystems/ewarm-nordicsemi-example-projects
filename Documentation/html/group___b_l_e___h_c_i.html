 Status codes specific to HCI

Detailed Description

Note:
Status codes in this file are used as return values and cause codes by GAP SVC interface, as defined by the GAP standard, and is therefore made available to the application, even though no HCI SVC interface is currently provided.
These codes are used as SVC parameters and in event reports, according to the BLE standard, with the values used "as is", NOT offset by NRF_HCI_ERR_BASE.
The codes are defined in the Bluetooth specification Version 4, Vol. 2, part D
