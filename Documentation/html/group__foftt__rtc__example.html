Detailed Description

This example enables the RTC with a TICK frequency of 8Hz. It also configures and enables the TICK- and the COMPARE0-interrupts. The compare interrupt handler will be triggered 3 seconds after the RTC starts. The TICK interrupt handler clears the interrupt request and toggles PIN8. The COMPARE0 handler clears the interrupt request an set PIN9 to 1.

Use board setup A for this example.
