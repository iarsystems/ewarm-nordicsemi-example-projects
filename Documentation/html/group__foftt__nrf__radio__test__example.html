Detailed Description

This example shows how to configure the radio for constant RX/TX carrier, modulated TX carrier and RX/TX sweep. The tests are controlled with simple one-letter commands via the serial port at 38400 baud. These radio parameters can be set at any time during the tests: output power, bit rate and channel. The time on each channel in sweep mode can also be set in steps of 1 ms from 1 ms to 99 ms.
