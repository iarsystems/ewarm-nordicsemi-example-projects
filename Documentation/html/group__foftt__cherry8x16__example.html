Detailed Description

This example will detect the characters by scanning the matrix for any state change (key(s) press or release). If any changes from the previous state is identified, then the received key codes are converted from hid to ascii which are transmitted over uart and can be listened by any COM listener like example:PUTTY (this is default behavior). However commenting out USE_UART can be done to to output modifier key presses to LEDs through GPIOs.

This example is meant to be used with the nRF6423 board. Connect nRF6310 P1 to LEDs. Connections between the two boards:

  • nRF6423 - nRF6310
  • (column lower) P3 - P0
  • (column higher) P4 - P1
  • (rows) P2 - P2

Can also be used with other keyboards, in which case row and column mapping have to be changed in this example

Pin number to use as UART TXD, LEDs for modifier key presses and UART cannot be used at the same time as both use SIMPLE_UART_TXD_PIN_NUMBER, so comment out UART code if using LEDs for modifier keys and vice-versa
