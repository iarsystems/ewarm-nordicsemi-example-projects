Detailed Description

Behaviour: This example generates variable duty cycle pulse train on selected GPIO pin. By default the pwm waveform generation will have a duty cycle of 1/256. If BUTTON1 is pressed the duty cycle is set to 224/256. If only BUTTON0 is pressed the waveform will set the duty cycle to 32/256.

Expected Results: In the default state LED0 should have a very weak light. When BUTTON1 is pressed the strenght of LED0 is increased to a bright light. When only BBUTTON0 is pressed LED0 will give a medium strenght light.
