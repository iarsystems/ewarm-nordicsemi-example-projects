Detailed Description

* Menu and configuration of radio variables ----------------------------------------------------------- Browse menu options by joystick up-down movement. Browse text of the option - Joystick right/left movement. Enter into a selection (enter to edit mode of selection) - push/click joystick. Increase value of selection - Joystick right movement. Decrease value of selection - Joystick left movement. Go back to menu after selection - push/click joystick.

Note:
: when the value of selection is changed in edit mode, its value is taken into effect immediately and the change in the radio spectrum with changed value can be noticed in the spectrometer
