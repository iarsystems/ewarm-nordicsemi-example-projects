Detailed Description

This project must be used together with the corresponding transmitter project for the PCA10001 board.

1) Flash button_radio_example in the PCA10001 board folder to PCA10001 board. 2) Flash button_radio_example in the PCA10000 board folder to PCA10000 board. 3) Use a terminal program on your PC to connect to the PCA10000 board. 38400 bps, 8 data bits, 1 stop bit. 4) Press a button on PCA10001 board, you should see "Button 0/1 pressed" on your terminal.

The receiver is configured to continuously receive packets from the corresponding transmitter project. The on-byte payload contains the button status on the transmitter board. If the byte is 1 Button 1 is pressed, if the byte is 2 Button 1 is pressed and if it's 3 both buttons are pressed.
