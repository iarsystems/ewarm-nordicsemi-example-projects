#define MAX_ITERATIONS_NEEDED_FOR_NEXT_BYTE   18
 Maximum iterations needed in the main loop between stop bit 1st byte and start bit 2nd byte. DTM standard allows 5000us delay between stop bit 1st byte and start bit 2nd byte. Byte transmission time is 10 * 1/19200 = approximately 5200 us/byte (including start/stop). The max time between two bytes is 10200us/625us/iteration = 17.3 iterations. This is rounded up to 18. If UART bit rate is changed, this value should be recalculated as well.
