Function Documentation

uint32_t app_button_init ( app_button_cfg_t *  p_buttons,
uint8_t  buttons_count,
uint32_t  poll_interval,
uint8_t  debounce_limit,
void *  p_buffer 
)

This function will initialize the specified pins as buttons, but it will not start the polling timer.

Note:
Normally initialization should be done using the APP_BUTTON_INIT() macro, as that will also allocate the buffer needed by the Buttons module.
Parameters:
[in]p_buttonsArray of buttons to be used (NOTE: Must be static!).
[in]buttons_countNumber of buttons.
[in]poll_intervalDuration between each time the buttons state is to be polled (in number of timer ticks).
[in]debounce_limitNumber of consequtive positive read operations before a button is reported as pushed.
[in]p_bufferPointer to memory buffer for internal use in the app_button module. The size of the buffer can be computed using the APP_BUTTON_BUF_SIZE() macro.
Returns:
NRF_SUCCESS on success, otherwise an error code.
