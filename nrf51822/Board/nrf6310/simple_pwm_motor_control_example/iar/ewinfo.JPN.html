Example description

  
Motor control example

DESCRIPTION
===========
 
   This example generates variable duty cycle pulse train on selected GPIO pin.
  By default the pwm waveform generation will have a duty cycle of 1/256.
  If BUTTON1 is pressed the duty cycle is set to 224/256.
  If only BUTTON0 is pressed the waveform will set the duty cycle to 32/256.

   In the default state LED0 should have a very weak light.
  When BUTTON1 is pressed the strenght of LED0 is increased to a bright light.
  When only BBUTTON0 is pressed LED0 will give a medium strenght light.
 
  For more detailed information see nRF51 SDK.

COMPATIBILITY
=============

   The example project is compatible with, on Nordic Semi nRF6310 - PCA10005 evaluation
  boards. By default, the project is configured to use the J-Link SWD interface.

CONFIGURATION
=============

   Make sure that the following jumpers and swiches are correctly configured on the
  nRF6310 evaluation board:
  Switches:
  S8  - VBUS
  S9  - ON
  S11 - ON
  
  PCA10005 board:
  Jumpers:
   K1 - 2-3
   P4 - 1-3 2-4 (32 kHz crystal)

  The Blinky application is downloaded to the iFlash.

GETTING STARTED
===============

  1) Build and download the program.

  2) Run the program.
